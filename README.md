# Templates

NodeDK templates.

Used by the create command in the `@nodedk/app` CLI.

### Usage

```sh
# Change to the template directory
cd basic

# Install packages
npm run install

# Run the app
npm run dev
```

Before pushing the changes, run this script from the templates root:

```sh
# Remove node_modules and package-lock.json files
./clean.sh
```
